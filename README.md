 🍔 Food Delivery Mobile App UI (Figma)

A modern and minimal **Food Delivery Mobile Application UI** designed in **Figma**. This project focuses on creating an intuitive and visually appealing user experience for ordering food online. The design includes authentication, home page, food browsing, shopping cart, and payment screens.

# 📖 Project Overview

The **Food Delivery Mobile App UI** is a high-fidelity mobile application prototype created using **Figma**. It demonstrates the complete user journey—from logging into the application to browsing food items, adding products to the cart, selecting payment methods, and placing an order.

The design follows modern UI/UX principles with a clean layout, soft colors, rounded components, and easy navigation.

# 🎯 Project Goals

- Design an attractive food ordering application.
- Create an intuitive user experience.
- Improve accessibility and ease of navigation.
- Develop a clean and modern mobile interface.
- Build a complete food ordering workflow.
